  chromeCastReceiverAppID

The ChromeCast Application ID of the receiver application to which you wish to cast.

@property (nonatomic) NSString *chromeCastReceiverAppID

Discussion

Set this property to a receiver’s Application ID before scanning for devices in order to restrict the scan to chromeCast devices that support the receiver. Leaving this value nil results in scanning for all ChromeCast devices.
