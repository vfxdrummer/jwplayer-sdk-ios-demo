JWAdConfig Class Reference

Inherits from NSObject
Declared in JWAdConfig.h

Overview

An object providing information about the way ads are handled by the player. Describes adMessage, skipMessage, skipText and skipOffset.

In current implementation adConfig object can be added to config and propagates to all adBreaks.
